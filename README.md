# Smart Attendance System

## Overview
Smart Attendance System is an AI-powered attendance management application that uses Face Recognition and Liveness Detection to automatically mark attendance. The system helps educational institutions and organizations reduce manual attendance tracking and improve accuracy.

## Features

- Face Detection using OpenCV
- Face Recognition based attendance
- Liveness Detection
- Automatic attendance recording
- Student registration module
- Dataset generation
- User-friendly interface
- Attendance report generation

## Technologies Used

- Python
- OpenCV
- NumPy
- Face Recognition
- Haar Cascade Classifier
